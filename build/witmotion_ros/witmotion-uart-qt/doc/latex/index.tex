\chapter{Witmotion Sensors UART Library}
\hypertarget{index}{}\label{index}\index{Witmotion Sensors UART Library@{Witmotion Sensors UART Library}}
\label{index_index}%
\Hypertarget{index_index}%
Core library allowing the external programs to read the data and control the parameters from the TTL/\+UART-\/compatible Inertial Measurement Units (IMU), manufactured by \href{https://www.wit-motion.com/}{\texttt{ Wit\+Motion Shenzhen Co.,Ltd}}.

The library implements the C++ API to program applications accessing Witmotion sensor devices supporting unified communication protocol. The class structure is defined to stay able to implement also any other protocol for the different hardware communication interface. In the current state, the library uses \href{https://qt.io}{\texttt{ Qt}} backend for UART/\+TTL serial interface through \href{https://doc.qt.io/qt-5/qtserialport-index.html}{\texttt{ Qt5 Serial Port}} internal API. Thus the {\bfseries{Qt library}} installation of version 5.\+1 or newer is the only system requirement for the library.\hypertarget{index_autotoc_md3}{}\doxysection{\texorpdfstring{Contents}{Contents}}\label{index_autotoc_md3}

The project strives to provide the following tools for every supported Witmotion device\+:
\begin{DoxyItemize}
\item Single-\/class based C++ API with dedicated dynamic library and set of header files;
\item Dedicated controller tool allowing the user to set the device parameters and view the produced data through convenient command-\/line interface.
