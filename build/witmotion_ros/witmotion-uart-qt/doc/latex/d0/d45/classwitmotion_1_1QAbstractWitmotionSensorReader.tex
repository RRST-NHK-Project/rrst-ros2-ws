\doxysubsection{Detailed Description}
This class allows the developer to write handler functions for multithreaded polling timer control, sensor configuration request and data decoding event. It also provides predefined signals to allow the end user to react on actual data acquisition and error occurence events. The common use case for this class is to provide a base interface for the family of sensors supporting the same protocol. Please refer to \doxylink{classwitmotion_1_1QBaseSerialWitmotionSensorReader}{QBase\+Serial\+Witmotion\+Sensor\+Reader} class for actual implementation of the currently supported UART-\/based protocol.
