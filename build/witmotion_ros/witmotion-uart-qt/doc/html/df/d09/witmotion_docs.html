Datasheets and official documentation

The library is developed according to the specifications released by Witmotion, the presented snapshot has a download date is 23.02.2022. The official website https://wiki.wit-motion.com is not always accessible, so the PDF snapshots are placed under IPFS web directory maintained by Elettra Sincrotrone Trieste.

  • JY901 9-Axis Combined IMU/Magnetometer/Altimeter Sensor
  • WT61C 3-Axis Combined Accelerometer/Gyroscope Sensor
  • WT31N 3-Axis Accelerometer/Gyroscope Sensor
  • WT901C 9-Axis Enclosed Inclinometer Sensor
  • WT901 9-Axis Open Circuit Inclinometer/Magnetometer Sensor
  • WT931N 9-Axis Inclinometer Sensor
  • HWT905 RS232/TTL 9-Axis IP67-Enclosed Combined IMU/Magnetometer/Altimeter/Inclinometer

As additional material, there is the command table, consisting of the list of special commands used to configure 9XX series of sensors.
